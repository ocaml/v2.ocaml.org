Download & Installation

OCaml is distributed by Inria. You will find below links to download the latest stable version, as well as links towards the SVN with the development version.

Stable Version

The most recent version is OCaml 3.12.1, released on 2011-07-04.

  • OCaml 3.12.1 Windows Installer (and its documentation)
  • OCaml 3.12.1 Sources (for compilation under Unix or Windows Cygwin)

Find more packages on Inria's website.

Development Version

You can access the most recent version in the SVN:

  • Browse the repository
  • Get a copy of the next release:

    svn checkout http:⁄⁄caml.inria.fr⁄svn⁄ocaml⁄trunk

Other Implementations

These projects allow you to compile OCaml code towards platforms traditionnally designed for other languages.

  • The js_of_ocaml project, an OCaml to Javascript compiler (stable)
  • The ocamljava project, an OCaml to Java compiler (experimental)
